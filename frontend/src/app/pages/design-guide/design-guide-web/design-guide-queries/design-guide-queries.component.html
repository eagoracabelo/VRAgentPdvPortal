Queries

O VRDesignGuide usa consultas de mídia para atender à grade escalonável, mas também tem uma lista de consultas para conveniência de estilo em dispositivos. As consultas são móveis primeiro, o que significa que elas segmentam min-width. Mobile-first As consultas são como a grade do VRDesignGuide é construída e é o método preferível de organizar o CSS. Significa tudo estilos fora de uma consulta de mídia se aplicam a todos os dispositivos e, em seguida, dispositivos maiores são direcionados para aprimoramento. Isso evita que pequenos dispositivos tenham que analisar toneladas de CSS não usadas. Os tamanhos para as consultas são:

  • Desktop HD: 1200px
  • Desktop: 1000px
  • Tablet: 750px
  • Phablet: 550px
  • Mobile: 400px
        

          /* Larger than tablet */
          @media (min-width: 750px) 

          /* Larger than desktop */
          @media (min-width: 1000px) 

          /* Larger than Desktop HD */
          @media (min-width: 1200px)
